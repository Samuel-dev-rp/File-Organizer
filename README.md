# File Organizer
A **simple organizer** from the **download** folder in **windows**.

## How work
The script identify the downloads folder and list all files. After list, it split the name of file to the extention and verify the subfolders, moving all files for the respective folder.

If the refer subfolder to extention not exists, the script make a directory with name to the extention.

## What i use in this project
- Python: The principal language from this project
- Extentions:
	- OS: A extention used for list and create directories
	- Shutil: A extention used for move all files to respective folder
